Bienvenue sur le site de Memory de l'ENI

Venez jouer avec nous et faites le meilleur score possible.

Règles du jeu :

  • Retournez 2 cartes. Si les images sont identiques, vous marquez 1 point.
  • Si les images sont différentes, les cartes se retournent face cachées.
  • La partie est terminé lorsque toutes les paires de cartes auront été retrouvées !

Si vous voulez vous souvenir de votre score. Inscrivez-vous via le formulaire.
Le jeu garde les 5 meilleurs scores en mémoire.
Si vous voulez relancer une partie, cliquez qur la barre d'espace
